h2(description). Description

This is a simple game. All you have to do is get to the exit!

h2. Running The Game

To run the game use the command:

@game.bat@

Or you can double click on game.bat

h2. Playing the Game

All controls are mentioned in the game, but here they are anyway:

Up - Jump
Left+Right - Move Left+Right
M - Toggle Music

h3. Get to the exit!

If you would like to play in fullscreen, open up config.yml and change:

@:fullscreen: false@

to

@:fullscreen: true@

h2. Music

The music is "UK Underground by MegaWatt-Man":http://www.newgrounds.com/audio/listen/280249
